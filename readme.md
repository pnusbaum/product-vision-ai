# Product Vision AI

Product Vision AI is an AI-powered product similarity search engine for e-commerce.

The system allows searching for visually similar products using:

- image-to-image similarity
- text-to-image similarity

It uses multimodal embeddings generated by CLIP, stored in PostgreSQL with pgvector, and deployed fully in Google Cloud Platform.

---

# Project goal

The goal of this project is to solve a real-world e-commerce problem:

> How to find similar products based only on an image or text?

Possible use cases:

- product recommendations
- duplicate detection
- competitor product matching
- catalog enrichment
- visual product search

---

# Features

## Search by image

User uploads an image.

System:

1. generates image embedding
2. compares it against product embeddings
3. returns most similar products

Endpoint:

```http
POST /search-by-image
```

## Search by text

User enters product description.

System:

1. generates text embedding
2. compares it against image embeddings
3. returns most relevant products

Endpoint:

```http
POST /search-by-text
```

# Deployment

## Backend

```bash

```

